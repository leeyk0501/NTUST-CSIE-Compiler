# NTUST-CSIE-Compiler
> 2020 Spring - Compiler Design
>

A simple compiler for the [Scala](https://en.wikipedia.org/wiki/Scala_(programming_language)) language.

## Project1 - Lexical Definition
Write a scanner for the Scala language in **lex**.

## Project2 - Syntactic and Semantic Definitions
Write a LALR(1) parser for the Scala language in **yacc**.

## Project3 - Code Generation
Generate code in **Java assembly language**.
